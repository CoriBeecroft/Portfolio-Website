4096

(Click here to play 4096!)

Description:

A remake of the game 2048, except instead of just manually moving tiles around to get to the 2048 tile, players have the option to specify an algorithm for moving tiles in order to get the 4096 tile.

Written With:

JavaScript, JQuery, HTML, CSS

Inspiration:

After playing 2048 for a while, I realized I was subconsciously developing a simple algorithm for playing the game. I was pretty sure I could come up with a good enough algorithm for gameplay that it could probably get far past the 2048 tile. This seemed like more fun to me than playing the game, so I made 4096.

Areas For Improvement:

  • GUI
    • Better editor for automated algorithm input
    • Show a list of helper functions available for writing automated algorithm
    • Speed control for automation
  • Make it so the game can be automatically run several times in a row and statistics about the runs will be shown(e.g. highest tile reached, how many moves it took, etc.)

Variable Level of Detail Essay

(Click here to see Variable Level of Detail Essay!)

Description:

This is one of my brother's projects that I helped out with. It is a tool for creating HTML essays that can be read at varying levels of detail. The idea here is that different readers of a particular piece of writing have different levels of knowledge and interest in the subject of the piece. This tool allows readers to choose how detailed the essay is while they're reading it. It's kind of like having in-line foot notes.

My Contribution:

  • Refactored a messy, convoluted code-base into something simpler and easier to use, which reduced to total line-count from 391 to 233
  • Fixed a major bug
    • Re-wrote most of the parser
    • Changed the structure of the essay nodes in the DOM from flat to hierarchical

https://github.com/westoncb/LODEssay
