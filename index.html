4096

(Click here to play 4096!)

Description:

A remake of the game 2048, except instead of just manually moving tiles around to get to the 2048 tile, players have the option to specify an algorithm for moving tiles in order to get the 4096 tile.

Written With:

JavaScript, JQuery, HTML, CSS

Inspiration:

After playing 2048 for a while, I realized I was subconsciously developing a simple algorithm for playing the game. I was pretty sure I could come up with a good enough algorithm for gameplay that it could probably get far past the 2048 tile. This seemed like more fun to me than playing the game, so I made 4096.

Areas For Improvement:

  • GUI
    • Better editor for automated algorithm input
    • Show a list of helper functions available for writing automated algorithm
    • Speed control for automation
  • Make it so the game can be automatically run several times in a row and statistics about the runs will be shown(e.g. highest tile reached, how many moves it took, etc.)

Doom Pet

Description:

A virtual pet game using graphics from id Software's original Doom games.

Written With:

Java, SGF(a Simple Game Framework written by my brother)

Inspiration:

I was looking for a project to work on between semesters one year and my brother suggested that I make a game using a simple game framework that he wrote. We figured it would be a good way for me to get some experience working with someone else's code and it would allow me to focus on some of the more interesting and fun parts of making a game. Though, I suspect he was just trying to get someone to test his framework because I ended up finding and fixing a lot of bugs while I was using it.
